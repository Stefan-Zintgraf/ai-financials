# ai-financials

Benchmark prompts for AI financial analysis across multiple models (ChatGPT, DeepSeek, Gemini, Kimi).

## Structure

- **chatgpt/** – Client suitability, equity valuation, portfolio rebalancing, stress testing, thesis critique
- **deepseek/** – DeepSeek model benchmark outputs
- **gemini/** – Gemini model prompts
- **kimi/** – Kimi model prompts
